说明：
     1.golang版本 1.10.3
	 2.安装MinGW64+gcc
	 3.非标准MVC框架
	 4.include文件夹和lib文件夹尽量位于根目录下与mian()函数同一根目录
	 5.布署时需要根目录下的config,log,include文件夹以及根目录下编译好的.exe文件
	 6.建议使用最新版VS Code作为调试工具
     7.如果编译通过但运行时提示找不到库文件，请查看用户环境变量和系统环境变量是否正确,尤其是gcc
	 8.上述方法如果无效，请将lib文件夹内的库文件直接放在项目根目录下尝试
